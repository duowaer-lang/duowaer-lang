<h1>Creating your First Storage Account within a Resource Group in Azure</h1>
This tutorial outlines the steps required to create your first storage account within a resource group in Azure.<br />

<h2>Environments and Technologies Used</h2>

- Microsoft Azure (Virtual Machines/Compute)

<h2>High-Level Deployment and Configuration Steps</h2>

- Step 1 Search Resource Groups and click on create at the top left of the page ( It will have a + sign) in Microsoft Azure
- Step 2 Make sure to add a name to your resource group and to choose the appropriate region
- Step 3 Search storage accounts and click on create a storage account in Microsoft Azure
- Step 4 Be sure to use the resource group you created in step 1 on the resource group option below subscription
- Step 5 Add a unique name to the storage account
- Step 6 Click on review + create
